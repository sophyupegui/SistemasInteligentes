# Sistemas Inteligentes

Repositorio con los trabajos y prácticas desarrollados en la materia **Sistemas Inteligentes**.  
El contenido se centra en la implementación y análisis de técnicas de inteligencia computacional aplicadas a problemas de ingeniería, utilizando principalmente **MATLAB** y **Simulink**.

## Contenidos principales

El repositorio incluye implementaciones y experimentos relacionados con:

- **Control difuso (Fuzzy Control)**
  - Diseño de controladores fuzzy
  - Evaluación del desempeño frente a distintos escenarios
  - Simulaciones y análisis de resultados

- **Metaheurísticas**
  - Algoritmos de optimización inspirados en la naturaleza
  - Ajuste de parámetros y comparación de soluciones
  - Pruebas experimentales y análisis de convergencia

- **Redes neuronales**
  - Modelado y entrenamiento de redes neuronales
  - Aplicaciones a identificación de sistemas y control
  - Evaluación del desempeño y resultados obtenidos

## Tecnologías utilizadas
- MATLAB
- Simulink
- NI LabView
- Microsoft Excel
